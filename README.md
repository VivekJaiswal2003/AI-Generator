# AI Article Generator

This project is a simple AI-powered article generator built using **n8n** and **Google Gemini AI**.

Users can enter any topic, and the workflow generates a well-structured article on that topic in real time.

## Features

- Generate articles on any topic
- Powered by Google Gemini AI
- Automated using n8n
- Clean and responsive web interface
- Easy to use

## Tech Stack

- HTML
- CSS
- JavaScript
- n8n
- Google Gemini AI

## How It Works

1. Enter a topic.
2. Click the **Generate Article** button.
3. The request is sent to an n8n webhook.
4. Google Gemini generates the article.
5. The generated content is displayed on the page.
